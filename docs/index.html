ZUSHAR WEB RESOURCE

zushar web resource is simply a RESTful web api that serves as a web service for zushar web tool for building forms online.
It's still under active development but currently on version 0.2.0 with users and forms module fully tested. The feature list and roadmap for the whole zushar eco-system will be included in this api documentation.

How to get started

it is very easy to start using the api:
First things first:

  • It is expected that your have installed or have latest version of postman
  • #Note:
    • In the cases that you don't have one please download it from chrome webstore
    • This means you must have chrome installed

      • Please note that this is not necessary the only limitation is that you will have to manually test the api using your own RESTful testing software. since during development postman was the best and available option; it was used to build a testing collection and is available for download. Just press the button postman collection above

  • Download the postman collections file.
  • open postman and import the collection json file you just download

    • Please note that this assumes you know your where abouts in using postman

  • A new collection will be created called zushar-api and you will get all the environment variables required for testing the api plus the routes and all the data needs set up for you.
  • Follow the readme from github to know how to install build and run the api

    Obviously this requires that you have mongodb installed in your system / computer.

  • Make sure the api is running.
  • Then just start exploring the api using the postman collection installed
  • its as simple as that
